# OLT Admin 文档

欢迎使用 OLT Admin！这是一个基于 React 19 + Vite + TypeScript 构建的现代化管理后台系统。

## 🚀 项目特色

- **现代化技术栈**: React 19、Vite、TypeScript
- **Feature-Sliced Design**: 高内聚、低耦合的架构模式
- **丰富的组件库**: 基于 Ant Design 和 shadcn/ui
- **完善的开发体验**: 热重载、类型安全、代码规范
- **企业级功能**: 权限管理、国际化、主题系统
